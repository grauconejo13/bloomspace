# Bloomspace 🌸

Bloomspace is a community wellness garden where users can draw digital flowers and attach encouraging messages for others to discover.

## Origins

Bloomspace began as a team project during nwHacks 2025 in Vancouver, Canada.

While the original hackathon prototype was never fully integrated or deployed, the project is being rebuilt and expanded as a full-stack application.

## Features

- Draw flowers using a digital canvas
- Attach positive messages
- Plant flowers in a shared garden
- Browse community creations
- Mobile responsive design

## Tech Stack

Frontend

- React
- Vite

Backend

- Node.js
- Express

Database

- MongoDB Atlas
